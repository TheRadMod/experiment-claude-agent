# Official Demo Applications

Source: https://github.com/anthropics/claude-agent-sdk-demos
Fetched: 2026-03-25

IMPORTANT: Demo applications by Anthropic for local development only. NOT for production use.

## Available Demos

### Email Agent
IMAP email assistant: display inbox, agentic search, AI-powered assistance.

### Excel Demo
Working with spreadsheets and Excel files using Claude.

### Hello World
Basic getting-started example for the Claude Agent SDK.

### Hello World V2
Examples for V2 Session API (unstable_v2_*): separate send()/stream()
with multi-turn conversation and session persistence patterns.

### Research Agent
Multi-agent research system:
- Breaks research into subtopics
- Spawns parallel researcher agents for web search
- Synthesizes findings into detailed reports
- Demonstrates subagent activity tracking

### AskUserQuestion Previews
Branding assistant with visual HTML preview cards for options.
Demonstrates previewFormat: "html" and WebSocket integration.

### Simple Chat App
React + Express chat UI with SDK, WebSocket streaming responses.

### Resume Generator
Generates .docx resume by web-searching a person's name and assembling findings.
